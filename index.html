    Keyboard Shortcuts

    • Backspace: Delete char before cursor
    • Arrow Keys: Navigate text editing cursor
    • Enter: Insert newline in code
    • Delete: Remove active shape
    • Tab: Cycle active shape
    • Ctrl+C: Toggle calibration and move shapes
    • Ctrl+F: Toggle fullscreen
    • Ctrl+L: Load map & hydra code configuration
    • Ctrl+M: Save map & hydra code configuration
    • Ctrl+Shift+E: Execute shape Hydra code
    • Ctrl+Shift+S: Toggle Hydra code visibility
    • Ctrl+Shift+M: Create new Hydra shape map
    • Ctrl+Shift+K: Open file upload (Audio/Video/Image)
    • Ctrl+Shift+I: Toggle loaded media
